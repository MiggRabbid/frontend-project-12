## Installation Instructions and System Requirements

#### Installation Instructions

```
git clone https://github.com/MiggRabbid/frontend-project-12
cd frontend-project-12
make install
make start

make localstart //for local host deployment
```

#### System Requirements

- Node.js v20.10.0
- npm 10.2.4
