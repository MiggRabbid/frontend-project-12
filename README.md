## Порядок установки и системные требования

#### Порядок установки

```
git clone https://github.com/MiggRabbid/frontend-project-12
frontend-project-12
make install
make build

make start //для запуска на локальном хосте
```

#### Системные требования

- Node.js v20.10.0
- npm 10.2.4
